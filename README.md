Descrição do trabalho:

  O trabalho consiste em um jogo arcade do tipo "jogo de reflexo".
  
  A ideia do jogo é de uma esteira com caixas em cima de uma esteira rolante, e na tela, tem zonas determinadas por linhas. Cada zona tem uma pontuação diferente, o quão mais rápido você apertar a tecla de espaço, maior a pontuação. Conforme o jogo vai passando, a velocidade de transição das caixas aumenta, aumentando assim, também, a dificuldade do jogo. O jogador tem 5 chances de erro, a cada fora da zona de pontuação, uma "vida" é gasta, e, ao fim das 5, o jogador perde a partida.
